# QA Python Sprint 6 - Автоматизация тестирования сервиса "Яндекс.Самокат"

## 🎯 Достижения проекта

### ✅ Реализованные тесты
- **FAQ раздел** - проверка корректности ответов на вопросы
- **Редиректы по логотипам** - переход на главную и Дзен
- **Оформление заказа** - полный флоу из двух точек входа

### 🛠 Технологический стек
- **Python 3.11** + **Pytest** - фреймворк тестирования
- **Selenium** - автоматизация браузера
- **Allure** - генерация отчетов
- **Page Object Pattern** - архитектура проекта

### 📁 Структура проекта
qa_python_sprint_6/
├── tests/ # Тестовые сценарии
├── pages/ # Page Object модели
├── locators/ # Локаторы элементов
├── data.py # Тестовые данные
├── urls.py # URL адреса
├── conftest.py # Фикстуры Pytest
└── requirements.txt # Зависимости проекта



### 🚀 Запуск тестов
```bash
# Все тесты с отчетом Allure
python -m pytest tests/ -v --alluredir=allure-results
allure serve allure-results

# Отдельные тест-сьюты
python -m pytest tests/test_faq.py -v
python -m pytest tests/test_order_page.py -v
